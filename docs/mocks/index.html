gi
gitinsights
UG

turn on cross-device sync?

your settings (theme, workweek, pto, layout) get written to a private gist in your own github account. nothing else syncs.

heads up: github only has one gist scope, and it covers all your gists. we only read/write our own (description = gitinsights:user-data:v1), but technically you're trusting us not to touch the rest.
  • commit data, diffs, and analytics never sync.
  • conflict rule: last-write-wins by timestamp.
  • turn it off any time. delete the cloud copy any time.
